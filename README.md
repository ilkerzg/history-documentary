# history-documentary

A Claude skill for producing fully narrated, illustrated, scored, word-synced-subtitled
history documentary films (3-40+ minutes) end to end on [fal.ai](https://fal.ai) — from
nothing but a topic and a duration.

- `SKILL.md` — the skill: the pipeline's five hard-won laws and the failure→cause map
- `references/style.md` — the image style contract (Seedream 5 Pro + style-reference images)
- `references/endpoints.md` — every fal endpoint used, with exact inputs and real limits
- `references/pipeline.md` — the step-by-step recipe, alignment math, verification protocol

Install: copy this folder into `~/.claude/skills/` (or add via your skills manager).
No scripts, no secrets — bring your own fal key at runtime.
